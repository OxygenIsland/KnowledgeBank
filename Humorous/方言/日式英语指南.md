## 1 日语只有5个元音，a（啊）、i（一）、u（呜）、e（安）、o（哦）
所以，所有的发音都要靠这5个元音和辅音进行组合
二锅头  =>  啊 go（姑哦）to（拖）
Mcdonald‘s  =>  嘛库多那鲁多
## 2 日语中还缺少一些辅音
F  =  h
family  =>  花米粒
V = b
vtamin => bitamin
 R、L
 light  =  right  =>  来一拖
## 3 日语中没有以辅音为结尾的音
cat  =>  cato看拖
dog  =>  dogu当谷
