## 🎨 AI 会做什么

### ✅ AI 自动完成的任务

1. **识别 Figma 链接** - 自动从你的消息中提取
2. **提取 File Key** - 从 URL 中解析出文件标识
3. **请求 Token**（仅首次）- 引导你获取访问令牌
4. **安装依赖** - 自动运行 `pip install requests`
5. **创建脚本** - 如果不存在，自动创建提取脚本
6. **执行提取** - 调用 Figma API 获取数据
7. **解析组件** - 识别所有 UI 组件及其类型
8. **推断接口** - 智能生成接口规格
9. **生成文档** - 创建完整的 Markdown 需求文档
10. **呈现结果** - 直接展示文档内容

### ❌ 你不需要做的事

- ❌ 手动运行 Python 脚本
- ❌ 安装任何软件（除了首次配置 Token）
- ❌ 编写任何代码
- ❌ 手动分析截图
- ❌ 整理文档格式

### ✅ 你只需要做的事

1. **提供 Figma 链接** - 粘贴设计稿的分享链接
2. **提供 Token**（仅首次）- 30秒内可完成
3. **确认推断结果** - 快速检查 AI 的识别是否正确
4. **提供业务细节**（可选）- 补充 AI 无法推断的业务逻辑

## 🔐 关于 Token 安全

### AI 如何处理你的 Token

✅ **安全措施：**
- Token 仅在当前会话中使用
- 不会被存储到任何地方
- 不会被记录到日志
- 仅用于调用 Figma 的官方 API
- 关闭对话后立即失效

✅ **Token 权限：**
- 只读权限（只能读取设计稿数据）
- 无法修改你的 Figma 文件
- 无法访问你的其他文件（除非你提供链接）

✅ **如果担心：**
- 可以生成一个专门用于此目的的 Token
- 使用完后可以在 Figma 设置中删除该 Token
- Figma 支持随时撤销 Token
